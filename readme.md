# Extern REPL 

Sublime 3 Plugin to send contents of Editor to a Console Window

currently works with [ConEmu](https://github.com/Maximus5/ConEmu)
on Windows (7 testet)

## Keymap

`Ctrl-1`:  sends current line to current Tab, cursor walks to next line
